
# IP Subnet Calculator

A modern, high-performance IP subnet calculator built with **Next.js**. This tool is designed to simplify network planning and subnetting calculations for IT professionals and students alike.

## Features

- **Subnet Calculation:** Instantly calculate network address, broadcast address, range of usable IP addresses, and total host capacity.
- **Modern Architecture:** Developed using the Next.js App Router for a fast, responsive user experience.
- **Clean UI:** Designed with simplicity in mind, focusing on accessibility and ease of use for technical tasks.
- **Open Source:** Modular design, making it easy to extend for custom networking requirements.

## Tech Stack

- **Framework:** Next.js (React)
- **Styling:** CSS/Tailwind CSS
- **Language:** TypeScript/JavaScript
